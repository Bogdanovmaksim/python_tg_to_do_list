# To-Do List Telegram Bot

Этот проект - Telegram-бот для управления списком задач (to-do list) с категориями, дедлайнами, поиском, экспортом и напоминаниями.

## Новые функции

- **Категории и дедлайны**: Добавляй категории и дедлайны при создании задач.
- **Поиск**: Ищи задачи по ключевым словам.
- **Экспорт**: Скачивай список задач в файл.
- **Напоминания**: Автоматические напоминания за день до дедлайна.
- **Инлайн-кнопки**: Удобные кнопки для отметки задач выполненными и удаления.

## Установка и запуск

1. **Клонируй репозиторий** или скачай файлы.
2. **Создай виртуальное окружение** (рекомендуется): `python -m venv venv` и активируй его.
3. **Установи зависимости**: `pip install -r requirements.txt`.
4. **Настрой .env файл**:
   - Добавь в проект `.env`.
   - Вставь свой токен бота: `BOT_TOKEN=ваш_токен`.
5. **Запусти бота**: `python main.py`.

## Структура

- `main.py`: Основная логика с обработчиками.
- `database.py`: Работа с SQLite (добавлены поля category, deadline).
- `scheduler.py`: Планировщик напоминаний (APScheduler).
- `requirements.txt`: Зависимости.
- `README.md`: Описание.

## Примечания

- `.env` файл не включай в репозиторий (он в .gitignore).
- Токен бота получай через @BotFather.
